Bildklassifikation mit ml5

Drag Image here

Hier sehen Sie das Ergebniss der Bildklassifikation mit dem TensorFlow Deep Learning Modell TensorFlow.js MobileNetV2. Das Modell trifft Vorhersagen darüber, was auf dem Bild zu sehen ist. Diese Vorhersagen wurden aus einem Datensatz von 50k Bildern vom Modell erlernt. Es werden die drei Erebnisse mit den höchsten Konfidenzwerten angezeigt.

Beispielbilder
Klicken sie auf die Beispielbilder um sie Klassfizieren zu lassen. Auf der rechten Seite können sie die Klassifikation sehen. Die Beispielbilder wurden so gewählt, dass die Hälfte von ihnen richtig und die andere Häfte falsch von dem Deep Learning Model klassifiziert wurden.
