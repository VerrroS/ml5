Fachliche Dokumentation

Ansatz

Um eine schnelle und einfache Lösung zu erstellen habe ich mich dazu entschieden die Klassifikation auf einer statischen WebPage zu implementieren. Ich nutze dabei kein Framework sondern schreibe nur HTML, CSS und JavaScript. Um redunganz im HTML zu vermeiden nutze ich Jinja Templates, welche ich dann mit einem Pythonscript in HTML umwandle. Da mir der Look und die Handhabung von Frameworks wie Plotly nicht gefällt und in diesem Falle eine grafische Darstellung mit css un Javascript einfach möglich ist, habe ich mich gegen ein Framework entschieden. Mit Css habe ich die volle Kontrolle über den Look und kann die Seite auch für mobile Geräte anpassen.

Aufbau

Zuerst lade ich die JavaScript Scripte in meine HTML Datei.

<script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/1.0.0/p5.min.js"></script> <script src="https://unpkg.com/ml5@latest/dist/ml5.min.js"></script> <script src="https://code.iconify.design/2/2.1.0/iconify.min.js"></script> HTML Content <script src="functions.js"></script> <script src="main.js"></script> ML5, P5 und Bootstrap sowie meine Css Dateien werden am Anfang der Seite geladen. Die JavaScripte main.js und functions.js werden am Ende der Seite geladen. Main.js beinhaltet die Hauptfunktionen und functions.js beinhaltet kleinere Funktionen.
